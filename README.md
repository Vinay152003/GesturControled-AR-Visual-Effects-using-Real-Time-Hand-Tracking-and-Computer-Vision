# Gesture-Controlled AR Visual Effects using Real-Time Computer Vision Hand Tracking

An AI-driven real-time AR visual effects system inspired by Naruto, built using computer vision-based hand tracking and gesture recognition to create interactive anime-style effects directly in the browser.

## 🚀 Features
- Real-time hand tracking using computer vision
- Gesture-controlled AR visual effects
- Anime-inspired (Naruto / Sasuke) power animations
- Runs directly in the browser (no installation required)

## 🛠️ How to Use
1. Download the project as `naruto-main.zip`.
2. Extract (unzip) the `naruto-main.zip` file.
3. Open the extracted folder.
4. Double-click `index.html` to open it in a modern web browser.
5. Allow camera access when prompted.

## 🎮 Controls
- **Right Hand Open:** Triggers Naruto-style power effect.
- **Left Hand Open:** Triggers Sasuke-style power effect.
- **Blue Hand Landmarks:** Visualizes real-time hand tracking detection.

## ⚠️ Note
- Glow effects were optimized/removed to ensure compatibility and smooth performance across devices.
- Best experienced in good lighting conditions for accurate hand detection.
